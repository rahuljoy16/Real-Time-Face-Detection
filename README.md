# Real-Time-Face-Detection
Can detect the various human facial expressions in real time via access to your camera

IMPORTANT : A common issue many people are running into is a 404 error when loading the models folder. This is caused when you are not running your code inside a server that loads the models folder. For example just right clicking the file and opening in Chrome will not work. I use Live Server with VSCode to run the code and that works for me.
